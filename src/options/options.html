Usage Instructions

  1. Start Ollama: Make sure Ollama is running on your local machine. Run ollama serve in your terminal.
  2. Pull the model: Ensure you have the translation model: ollama pull translategemma:4b
  3. Select text: Highlight any text on a webpage that you want to translate.
  4. Translate: Use one of these methods:
    • Right-click and select "Translate with LingoBridge"
    • Press Alt+Shift+T
    • Click the LingoBridge toolbar icon
